# Setup

1. Import the exported workflows from `n8n-workflows/`.
2. Configure credentials in n8n.
3. Set required environment variables.
4. Connect external services:
   - Pipedrive
   - Gmail
   - Google Calendar
   - Google Drive
   - Twilio
   - Firecrawl
5. Enable the required webhooks.
6. Test each workflow independently before running the full pipeline.
