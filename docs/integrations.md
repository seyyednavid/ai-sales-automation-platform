# Integrations

## Pipedrive
Used as the CRM for storing contacts, organizations, leads, and deals.

## Gmail
Used to create personalized outreach email drafts.

## Google Calendar
Used to check availability and schedule product demos.

## Google Drive
Used as the source for uploading the Ideal Customer Profile document.

## Twilio
Used to provide the business phone number for inbound calls.

## Firecrawl
Used to search and extract prospect information from public web sources.
